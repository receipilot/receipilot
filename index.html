🚀 פרסום ל-GitHub Pages — 5 דקות
⚠ למה GitHub Pages? OAuth של Google ו-Microsoft חוסם כתובות file://. ב-GitHub Pages האפליקציה אוותו לך URL קבוע שעובד עם OAuth — כמו כל אתר.
1
צור חשבון GitHub חינמי
גש ל-github.com → Sign up. שם משתמש יהפוך לחלק מה-URL שלך.
2
צור Repository חדש
GitHub → New repository → שם: receipilot → Public → Create
3
העלה את הקובץ
בדף ה-Repository → לחץ uploading an existing file → גרור את ReceiPilot-OAuth.html → שנה שם ל: index.html → Commit
4
הפעל GitHub Pages
Settings → Pages → Source: Deploy from branch → Branch: main → Save
ה-URL שלך יהיה:
https://YOUR-USERNAME.github.io/receipilot/
5
עדכן Google Cloud Console
ב-Credentials → ערוך OAuth Client → הוסף ל-Authorized origins ו-redirect URIs:
https://YOUR-USERNAME.github.io
https://YOUR-USERNAME.github.io/receipilot/
6
עדכן Azure Portal (Outlook)
App registrations → Authentication → הוסף Redirect URI:
https://YOUR-USERNAME.github.io/receipilot/
🔗 הזן שם משתמש GitHub שלך
